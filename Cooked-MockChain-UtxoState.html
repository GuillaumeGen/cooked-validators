newtype UtxoState Source #

A UtxoState provides us with the mental picture of the state of the UTxO graph: Each address has a set of UTxOs that consist in a value and some potential datum.

newtype UtxoValueSet Source #

Represents a set of values, yet, we use a list instead of a set because Value doesn't implement Ord and because it is possible that we want to distinguish between utxo states that have additional utxos, even if these could have been merged together.
